# 使用指南

## 快捷键

| 快捷键                | 作用           |
| --------------------- | -------------- |
| shift+F12             | 默认化布局     |
| alt+数字              | 切换对应的窗口 |
| 进入project窗口后左右 | 打开关闭文件   |
| crtl+r				| 替换			 |
| crtl+f				| 查找			 |
| crtl+shift+I			| 看源码		 |
